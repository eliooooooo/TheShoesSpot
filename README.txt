# The Shoes Spot (projet fictif)

Ce projet à été réalisé dans le but de manipuler un site construit sur une base de donnée, de travailler avec les templates twig, d'imaginer les parcours utilisateurs du'un site e-commerce.  
  
Il a été réalisé en deux semaines, dans le cadre de mes études.
